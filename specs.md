# ShowFreak - Product Requirements

## Overview
ShowFreak is a web application that allows users to discover movies and TV shows, track what they have watched, and receive personalized recommendations.

## User Goals
- Discover new movies and TV shows
- Track watched content
- Save favorites and wishlist items
- Receive personalized recommendations

## Core Features

### 1. Browse & Search
- Users can search for movies and TV shows
- Users can view detailed information:
  - Title
  - Release date
  - Genres
  - Actors
  - Rating (IMDb or similar)

### 2. Personal Library
- Users can:
  - Mark content as watched
  - Add to favorites
  - Add to wishlist
- Users can:
  - Rate content
  - Add personal notes

### 3. Library Management
- Users can:
  - Search within their library
  - Filter by genre
  - Sort by:
    - Release year
    - External rating
    - Personal rating

### 4. Preferences
- Users can dislike content
- The system learns from:
  - Watched content
  - Ratings
  - Dislikes

### 5. Recommendations
- Users receive personalized suggestions
- Recommendations can be:
  - Based on user behavior
  - Based on a specific movie (e.g., similar to Avatar)

## Non-Goals (for MVP)
- No social features
- No complex AI model training (use external APIs or simple logic)
